# 🚀 ЗАПУСК ЗА 10 МИНУТ

## ШАГ 1 — Загрузи файлы на GitHub

1. Зайди на https://github.com
2. Нажми зелёную кнопку **"New"** (новый репозиторий)
3. Название: `ai-voice-agent`
4. Нажми **"Create repository"**
5. Нажми **"uploading an existing file"**
6. Перетащи ВСЕ файлы из папки `ai-voice-agent` (кроме `node_modules`)
7. Нажми **"Commit changes"**

---

## ШАГ 2 — Деплой на Railway

1. Зайди на https://railway.app
2. Нажми **"Start a New Project"**
3. Выбери **"Deploy from GitHub repo"**
4. Выбери репозиторий `ai-voice-agent`
5. Railway начнёт деплой автоматически ✅
